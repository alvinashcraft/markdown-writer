# Microsoft Store Submission — QuietMark v1.0.0.0

## Store Listing

**Title:** QuietMark

**Short Title:** QuietMark

**Voice Title:** Quiet Mark

**Short Description:** A distraction-free markdown editor with live preview.

**Description:**

QuietMark is a clean, focused markdown editor designed for writers who want to see their work rendered in real time without clutter. Type on the left, preview on the right — no setup, no fuss.

Built with a native Windows look and feel using Fluent UI and Mica materials, QuietMark fits right in on Windows 10 and 11. It supports opening, editing, and saving markdown files (.md, .markdown, .txt), plus exporting to HTML or PDF when you're ready to share.

Key highlights:
- Split-pane editor with instant live preview powered by CodeMirror 6
- Dark and light themes with automatic system detection
- Full file operations: New, Open, Save, Save As with unsaved-changes protection
- Remembers your last opened file between sessions
- HTML and PDF export from the File menu
- Word count, character count, and selection stats in the status bar
- Standard keyboard shortcuts (Ctrl+S, Ctrl+O, Ctrl+N, Ctrl+B, Ctrl+I)
- Available in English and Spanish

Whether you're drafting a README, writing blog posts, or taking notes, QuietMark keeps things quiet so you can focus on what matters — your words.

## Testing Details (for Store reviewers)

QuietMark is a standalone desktop markdown editor that does not require an account, login, internet connection, or any external services.

**To test the app:**

1. Launch QuietMark. On first launch, a welcome markdown document is displayed in the editor (left pane) with a live preview (right pane).
2. Edit the text in the left pane — the right pane updates in real time.
3. Use **File > Save** (Ctrl+S) to save as a `.md` file. The default save location is the user's Documents folder.
4. Use **File > Open** (Ctrl+O) to open any `.md`, `.markdown`, or `.txt` file.
5. Use **File > New** (Ctrl+N) to start a blank document.
6. Close and reopen the app — it will reopen the last saved file automatically.
7. Toggle dark/light mode via the toolbar sun/moon button or **View > Toggle Dark Mode** (Ctrl+Shift+D).
8. Test export via **File > Export as HTML** (Ctrl+Shift+E) and **File > Export as PDF** (Ctrl+Shift+P).
9. Toolbar formatting buttons apply **bold** (Ctrl+B) and *italic* (Ctrl+I) to selected text.
10. The status bar at the bottom shows the current file name, word count, and character count.

**No special configuration or credentials are required.**
